ABOUT

FreeSearch is a prototype of a search engine alternative that does not give linear ranked bias to its results as Google and all other search engines do. No result is given priority or value over the others and all results appear the same (as a circle) and unordered (in a random spread) when deciding which to investigate. This allows for the user to decide which seems most valuable for their purposes and challenges them to investigate past 1st or 2nd traditionally ranked results. This piece was inspired by the shift of user agency demonstrated in THE WEB STALKER
